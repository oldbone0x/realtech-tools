# RealTech Tools

🧰 AI-powered micro-tools for developers and creators.

**Live Site:** https://tools.real-tech.online

---

## 🏗️ Architecture

- **Single Domain:** `tools.real-tech.online`
- **Hosting:** Cloudflare Pages
- **Stack:** React + Vite + TypeScript + Tailwind CSS
- **Deployment:** Fully automated via GitHub Actions

---

## 🚀 Quick Start

### Development

```bash
npm install
npm run dev
```

Visit http://localhost:3000

### Build

```bash
npm run build
```

Output: `dist/`

### Deploy

```bash
./deploy_cf_pages.sh realtech-tools
```

---

## 📦 Adding a New Tool

### 1. Create Tool Component

```bash
mkdir -p src/tools/[tool-name]
```

**File:** `src/tools/[tool-name]/index.tsx`

```tsx
import React from 'react';

export const [ToolName]: React.FC = () => {
  return <div>Tool UI</div>;
};
```

### 2. Register in realtech-tools.json

```json
[
  {
    "name": "Tool Name",
    "slug": "tool-name",
    "description": "Description",
    "path": "/tool-name",
    "category": "developer"
  }
]
```

### 3. Add to Router

**File:** `src/App.tsx`

```tsx
import { [ToolName] } from './tools/[tool-name]';

// In Routes:
<Route path="/tool-name" element={<[ToolName] />} />
```

### 4. Build and Deploy

```bash
npm run build
./deploy_cf_pages.sh realtech-tools
```

**Done!** Tool is live at `https://tools.real-tech.online/tool-name`
